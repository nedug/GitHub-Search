# React searching repos on GitHub

***

* search GitHub repositories by username (+debounce)
* display list of repositories
* add/remove favorites
* saving data to localStorage

***

```
Technologies and libraries in development:
- React
- TypeScript
- Redux Toolkit Query
- Tailwind
- React-router-dom
- Debounce
```
